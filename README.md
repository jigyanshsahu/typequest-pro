# TypeQuest Pro

A premium typing practice app that helps you learn and revise topics while improving your typing speed. Powered by local AI.

## Technologies Used

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Getting Started

Follow these steps to run the project locally:

```sh
# Step 1: Clone the repository
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory
cd typequest-pro

# Step 3: Install dependencies
npm i

# Step 4: Start the development server
npm run dev
```
